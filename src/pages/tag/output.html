Exemplo

Realiza um cálculo e mostra o resultado numa tag <output>:


    <form oninput="x.value=parseInt(a.value)+parseInt(b.value)">0
        <input type="range" id="a" value="50">100
        +<input type="number" id="b" value="50">
        =<output name="x" for="a b"></output>
    </form>

Definição e Uso

A tag <output> representa o resultado de um cálculo (realizado por um script por exemplo).

Atributos

Atributos Valor Descrição
for element_id Especifica o relacionamento entre o resultado do cálculo e os elementos usados no cálculo
form form_id Especifica qual formulário a output pertence
name name Especifica o nome para a output
