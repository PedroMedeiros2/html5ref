Exemplo

Uma tabela HTML simples, com duas células da tabela: 


<table>
    <tr>
        <td>Cell A</td>
        <td>Cell B</td>
    </tr>
</table>

Definição e Uso

A tag <td> define uma célula padrão em uma tabela HTML.
Uma tabela HTML possui dois tipos de células:

  • Células do cabeçalho - contém informações do cabeçalho (criadas com o elemento <th> )
  • Células padrão - contém dados (criados com o elemento <td>)

O texto em <th> elementos está em negrito e centralizado por padrão.
O texto nos elementos <td> é regular e alinhado à esquerda por padrão.
