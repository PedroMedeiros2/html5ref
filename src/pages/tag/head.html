Exemplo

Um documento HTML, com a tag <title> dentro da seção head:

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
</head>
<body>
The content of the document......
</body>
</html>

Definição e Uso

A tag <head> é um container para todos elementos do cabeçalho.
A tag <head> pode incluir um título para o documento, scripts, estilos, informações meta, dentro outras.
Os seguintes elementos podem ser colocados dentro da tag <head>:

  • <title> (Esse elemento é requerido em um documento HTML)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
